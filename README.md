# Gulp Template
This is a gulp template I am currently using for web development.

## Feature
- Minify javascript, css and images

## Install
1. Install node.js
2. Install gulp
```
 npm install -g gulp
```
3. Install gulp packages
```
 npm install
```

## Folder Structure
- Source : /src
- Destination : /dest

## Usage

###  Run
```
gulp
```
Access development web via http://locahost:8080

### Build

```
gulp build
```
